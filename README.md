# Project Risk Prediction using Machine Learning

This project develops a data-driven approach to predict project risk exposure using Logistic Regression. A publicly available Kaggle dataset was used, covering key project features such as budget, team size, and stakeholder factors.

## Key Steps
- Data Cleaning and Preprocessing  
- Exploratory Data Analysis (EDA)  
- Model Development (Multi-class & Binary Classification)  
- Model Evaluation (Accuracy, Precision, Recall, F1, ROC-AUC)  
- Feature Importance Analysis  

## Results
- Multi-class model: Moderate performance  
- Binary model: Strong performance (F1-score ≈ 0.90)  
- Key risk drivers identified (e.g., stakeholder availability, team experience)

## Tools & Libraries
Python was used as the core programming language, with Pandas and NumPy for data processing. Visualisation was done using Matplotlib and Seaborn. Machine learning modelling, tuning, and evaluation were carried out using Scikit-learn, while KaggleHub was used for dataset access.
